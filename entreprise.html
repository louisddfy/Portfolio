1

Conception d'un Dashboard de Pilotage (PME)

De la donnée brute au visuel

Le Challenge Technique

Le client disposait de données fragmentées : une partie des ventes dans une base SQL, et les objectifs commerciaux dans des fichiers Excel dispersés. Il était impossible d'avoir une vision unifiée.

ETL & Modélisation

Fusion des sources hétérogènes (Jointures SQL et concaténation Qlik) pour créer un modèle de données cohérent (Étoile).

Data Visualization

Création d'indicateurs (KPIs) dynamiques et ergonomiques pour la direction.

Environnement Technique

Qlik Sense SQL Server Excel (CSVs)

Résultat : Gain de temps estimé à 2h/semaine pour le client dans la génération de ses rapports.

Preuves de réalisation

Back-End : Structuration des tables

Front-End : Restitution visuelle pour le client

2

Optimisation & Debugging

Maintenance corrective et évolutive

Intervention sur des applications existantes jugées trop lentes par les utilisateurs. Analyse des scripts de chargement et des expressions de calcul.

Actions concrètes :

  • Refactoring de code : Nettoyage des boucles inutiles dans le script de chargement.
  • Set Analysis : Optimisation des formules d'agrégation pour réduire le temps de calcul CPU.
  • Debugging : Correction d'écarts de chiffres via l'analyse des logs.
